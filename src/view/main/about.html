About Colosimo

Colosimo was an early Chicago Boss, and likewise the Colosimo project is an example of an early project with the Chicago Boss framework.

Colosimo displays templates, styles, PostgreSQL db shards, user registration/login, and bcrypt password encryption.

For more details on getting started with Chicago Boss, checkout An Evening with Chicago Boss, or visit the mailing list.

For details on Colosimo, checkout the github repo.
